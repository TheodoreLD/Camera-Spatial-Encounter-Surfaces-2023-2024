# Results Included In This Repository

The `results/` directory contains compact final outputs for the two 2024 models.
It is meant for review and interpretation, not as a complete generated-output
archive.

## Not Included By Default

The full generated output folders contain GeoTIFF rasters, full prediction-grid
CSV files, and many additional diagnostic figures. These are omitted here to
keep the GitHub project compact. They can be regenerated by rerunning the final
scripts.
